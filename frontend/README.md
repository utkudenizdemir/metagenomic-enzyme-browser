# Frontend (Next.js)

This directory contains the frontend component of the original metagenome browser framework.

The frontend is built using Next.js, TypeScript, and Tailwind CSS, and interacts with the Flask backend via API endpoints.

## Notes

* The frontend structure was not the primary focus of my MSc project work
* My contributions mainly focused on backend development, data processing, and database design
* Only minor adjustments may have been made to support backend integration

## Running the frontend

```bash
npm install
npm run dev
```

Then open:

```
http://localhost:3000
```
